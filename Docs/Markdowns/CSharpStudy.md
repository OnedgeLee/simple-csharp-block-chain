## Naming Convention

- Namespace
  - PascalCase
- Enum
  - PascalCase
- Class
  - PascalCase
- Method
  - PascalCase
- Property
  - PascalCase
- Interface
  - IPascalCase
- Argument
  - camelCase
- Local variable
  - camelCase
- Public member variable
  - PascalCase
- Private member variable
  - _camelCase
